# SM-W lite+ Log Analyzer

Python tool to analyze log files generated by the SM-W lite+ (IE Tecnologia) electrical monitoring device.

## Features

- Parse logs (RMS voltage, RMS current, active power, accumulated energy)
- Auto-scale raw values (×100 → real-world units)
- Generate time-series plots:
  - `tensao.png` — Voltage [V]
  - `corrente.png` — Current [A]
  - `potencia.png` — Power [W]
- Compute daily energy usage (kWh/day) from accumulated `epa_c`
- Detect outliers using the IQR method (voltage, current, power)
- Automatic text report with statistics, kWh/day and examples of outliers
- Interactive CLI for folder selection and report options

## Installation

```bash
pip install numpy pandas matplotlib
```

## Usage

```bash
python analite.py
```

The script prompts for:

1. Source folder (`./logs` or a custom path)  
2. Report generation mode (terminal only, file only, or both)  

## Log Format

```text
hour:minute:second:pa:epa_c:epa_g:iarms:uarms
```

Values `pa`, `iarms`, and `uarms` are scaled by ×100 by the SM-W lite+; the script converts them to W, A, and V respectively.
